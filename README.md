# SriVoraTech - Next-Generation IT & Tech Solutions Platform

Welcome to **SriVoraTech**, an enterprise-grade digital services platform delivering cutting-edge software development, AI & cloud solutions, cybersecurity, and digital transformation.

---

## 🌟 Key Features

- **Modern UI/UX Architecture**: Crafted with React 18, Vite, dynamic glassmorphism, and responsive design components.
- **Core IT Services Hub**: Highlighting Custom Software Development, Cloud & DevOps, Cybersecurity, Mobile & Web Applications, and AI Solutions.
- **Interactive Career Portal**: Dynamic job listings with direct application submission, file/resume upload support, and automated storage.
- **Client Inquiry & Contact Hub**: Interactive consultation request forms with Google Sheets integration for lead & inquiry tracking.
- **Backend API Server**: Node.js + Express backend handling secure file uploads, email/spreadsheet syncing, and inquiry API routes.

---

## 🚀 Tech Stack

- **Frontend**: React 18, Vite, Lucide Icons, Modern Vanilla CSS Design System
- **Backend**: Node.js, Express, Cors, Multer
- **Database / Integrations**: Google Sheets API (via `google-spreadsheet`), Dotenv
- **Tooling**: Node.js, NPM

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/srivoratech9/srivoratech.git
   cd srivoratech
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Copy `.env.example` to `.env` and fill in any required variables (e.g. Google Sheets API credentials).
   ```bash
   cp .env.example .env
   ```

4. **Start the Frontend Development Server**:
   ```bash
   npm run dev
   ```

5. **Start the Backend Express Server**:
   ```bash
   npm run server
   ```
